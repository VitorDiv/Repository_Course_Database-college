Objetivo:
Criar um banco de dados para gerenciar as informações de uma clínica médica, incluindo dados sobre clínicas, consultas, exames, 
históricos médicos, pacientes, planos de saúde e secretarias.

 
* Requisitos:
* Criação do Banco de Dados:

Criar um banco de dados chamado clinica_medica.
* Criação das Tabelas:
* Criar as seguintes tabelas com as respectivas colunas:

-Clínica: Para armazenar informações sobre as clínicas (nome, endereço, telefone, especialidades oferecidas).

-Consulta: Para registrar informações sobre as consultas (data, paciente, médico, diagnóstico, tratamento recomendado).

-Exames: Para registrar os exames solicitados durante as consultas (tipo de exame, data, resultados).

-Históricos Médicos: Para armazenar o histórico médico de cada paciente (informações anteriores de saúde, doenças preexistentes, tratamentos realizados).

-Paciente: Para armazenar dados dos pacientes (nome, CPF, data de nascimento, endereço, telefone).

-Plano de Saúde: Para registrar os planos de saúde dos pacientes (nome do plano, validade, cobertura).

-Secretaria: Para armazenar informações da equipe de secretárias da clínica (nome, telefone, email, função).

* Relacionamentos:
* Estabelecer relacionamentos entre as tabelas, como:

-Paciente pode ter várias consultas, exames e históricos médicos.

-Cada consulta está associada a um paciente e um médico.

-O plano de saúde está vinculado a cada paciente.

-A secretaria é responsável por agendar as consultas e gerenciar os pacientes.

*Operações Básicas:

-Inserir dados iniciais nas tabelas.

-Realizar consultas simples, como buscar pacientes por nome, listar consultas de um médico, exames solicitados, etc.
